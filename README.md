# Nonebot_Plugin_MCplayer
  
_✨ 基于OneBot适配器的[NoneBot2](https://v2.nonebot.dev/)查看MC服务器在线玩家信息与服务器状态的插件 ✨_
  
</div>

## 功能

- 查看MC服务器在线玩家信息与服务器状态

## 安装

- 使用 pip

```
pip install nonebot_plugin_mcplayer
```
